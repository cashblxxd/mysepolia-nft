# MySepolia Sample Polygon NFT Contract

## Prerequisites

- [Node.js](https://nodejs.org/en/) (>= 10.16.0)

## Deployment

1. Install project

```bash
npm i
```

2. Create an `.env` and add your MetaMask `PRIVATE_KEY`

3. Deploy with

```bash
npx hardhat compile
npx hardhat run scripts/deploy.js
```
